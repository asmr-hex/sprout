# Universal Plug and Play (UPnP)
this is a set of networking protocols facilitating "zero configuration" networking between devices.

### stack
UPnP consists of 6 layers:
1. Discovery
2. Description
3. Control
4. Eventing
5. Presentation
6. Addressing (_optional_)
